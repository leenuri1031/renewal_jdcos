행사 기간 : 2025년 3월 12일 ~ 2025년 3월 25일
 (* 조기 소진 시 종료됩니다.)

행사 대상 : 이벤트 기간 내 회원가입 후 '쥬디메르 더미셔틀 리프팅 키트 1x1' 구매 고객 한정
 - 적립금 페이백은 회원 전용 혜택입니다. 비회원의 경우 회원가입 후 이용 바랍니다.
  [회원가입 바로가기]

행사 경품 :
 - 구매 후 만족 시 100% 적립금이 지급됩니다. 이벤트 상품 구매금액 100%가 적립금으로 페이백 됩니다. (적립금은 더미셔틀 리프팅 키트로 구성된 상품 한정으로 사용 가능합니다.)
 - 적립금 사용 기간은 발급일로부터 30일 이내입니다.
 - 구매 후 불만족 시 100% 환불 됩니다.
 (* 단, 7일 이내 환불 요청이 없을 시 적립금이 지급됩니다.)
 - 적립금은 쿠폰 형태로 발급되며 개별 회원의 쿠폰함에 39,000원 할인 형태로 노출됩니다.
 - 회원 가입 축하금 및 카카오톡 전체 동의 쿠폰은 본 이벤트 상품에는 적용되지 않습니다.
 단, 쥬디메르의 다른 상품에는 즉시 사용 가능합니다.

적립일 : 구매일로부터 7일이 경과한 익일(다음 날) 자동 지급 (7일 후 다음 날이 주말 또는 공휴일인 경우, 그다음 영업일에 지급)
  예시)
    ✔ 3월15일(토) 구매 → 3월 21일(금)경과 : 3월 24일(월) 지급
    ✔ 3월16일(일) 구매 → 3월 22일(토)경과 : 3월 24일(월) 지급
